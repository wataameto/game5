# びよんびよんボールキャッチャー3D (Bouncy Ball Catch & Auto-Throw) ☄️🤖✨

カメラの視点が回転せず、アリーナ全体を見渡せる **固定俯瞰三人称視点 (Non-rotating Chase Camera)** を採用した、ハイスピード 3D 物理アクションゲームです。

👉 **[ゲームを今すぐ遊ぶ！ (Live Demo)](https://wataameto.github.io/game5/)**

---

## 🎮 操作方法 (Controls)

視点は固定されているため、キーボードの方向キーを押した方向にそのまま直感的に走ることができます。

| キー入力 | アクション |
| :--- | :--- |
| **`↑` / `W`** | 画面の**奥 (北)** へ進む |
| **`↓` / `S`** | 画面の**手前 (南)** へ進む |
| **`←` / `A`** | 画面の**左 (西)** へ進む |
| **`→` / `D`** | 画面の**右 (東)** へ進む |

## 🌟 ゲームの特徴 (Features)

### 1. 🎥 ぐるぐる回らない「酔わない」三人称カメラ
- プレイヤーの回転に合わせてカメラがぐるぐる回るのを完全に廃止しました。
- アリーナに対してカメラの角度が常に一定に固定されており、プレイヤーの移動（`X` と `Z`）にのみ滑らかにスライド追従します。これにより、3Dゲーム特有の画面酔いを防ぎ、アリーナ全体を常に把握しやすくなっています。

### 2. 🫴 体当たり自動「投げまくり」システム
- 面倒な照準合わせやキー入力は不要！アリーナをびよんびよん跳ね回るでっかいボールに**自ら体当たり（接触）するだけ**でアクションが発動します。
- プレイヤーとボールが激突した瞬間、ボールを手に引き寄せ、**アリーナ内の最も近くにいるゼリーモンスターを自動でロックオンして「シュバババッ！」と高速で投げつけます**。
- 跳ね返ってきたボールを再び体当たりで拾っては自動射出する、ドミノ式の連鎖的な爽快アクションが楽しめます。

### 3. 🥎 モンスター撃破 ＆ ボール増殖
- アリーナの外から、プレイヤーを狙ってジワジワ迫ってくる緑のゼリーモンスターが出現します。
- 投げつけたボールを命中させて倒すとスコアが加算されます。
- さらに、モンスターを倒した際に**約35%の確率で新しいボールがアリーナにポップ（最大6個まで増殖）**します。ボールが増えるほど、体当たり回収と自動スローが加速し、アリーナ中がびよんびよん跳ねるボールでお祭り状態になります！

## 🛠️ ローカルでの起動方法 (Local Development)

### 1. 依存関係のインストール (テスト用)
```bash
npm install
```

### 2. ローカルサーバーの起動
```bash
npx http-server -p 8085
```
